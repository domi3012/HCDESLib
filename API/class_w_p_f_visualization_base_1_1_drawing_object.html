Detailed Description

A base drawing object that can be added to and moved on DrawingOnCoordinateSystem Actual drawing shapes are implemented as a Path drawing object, the drawing should be done in the constructor using a local coordinate system with (0,0) origin. Do not use the start position as a drawing reference point. Drawings will be automatically shifted to that position
