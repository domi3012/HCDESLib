Creates a visualization of a simulation model at a given time. For each control unit of the model it is checked if a BaseWPFControlUnitVisualizationEngine has been specified and if so it is used to create the dynamic visualization of the control. If not nothing is done for that control unit.

void SimulationWPFVisualizationTools.BaseWPFModelVisualization.InitializeModelVisualizationAtTime ( DateTime  initializeTime,
SimulationModel  simulationModel 
)
inline

Initilializes the visualization, can be at start of simulation (e.g. static visualization), at re-enabling visualization during a visualization run or re-runs after completion. For each control unit of the model it is checked if a BaseWPFControlUnitVisualizationEngine has been specified and if so it is used to create the static visualization of the control. If not nothing is done for that control unit.

Parameters
initializeTimeThe time the model is initialized
simulationModelThe model at the initilization time
